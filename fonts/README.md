# Fonts

字体文件放在这个目录。当前示例配置使用：

- `ZhiMangXing-Regular.ttf`：当前示例中文字体
- `Caveat-wght.ttf`：当前示例英文字体

其他已下载字体：

- `chinese_handwriting.ttf`：LXGW WenKai，干净硬笔风格，但更规整
- `LongCang-Regular.ttf`：日常手写感，仍然略方
- `LiuJianMaoCao-Regular.ttf`：更草，辨识度较低
- `MaShanZheng-Regular.ttf`：更接近毛笔书法
- `english_handwriting.ttf`：Patrick Hand

字体许可证和版权声明见 [LICENSES.md](LICENSES.md)。完整 OFL-1.1 文本见 [OFL-1.1.txt](OFL-1.1.txt)。

如果配置中指定的字体文件不存在，CLI 会给出明确错误。
